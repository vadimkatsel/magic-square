# Magic Square

An interactive web application for generating and exploring magic squares.

A magic square is a square grid of numbers where the sum of every row, every column, and both main diagonals is equal. This project provides a simple visual way to create and experiment with them directly in the browser.

🌐 Live Demo: https://vadimkatsel.github.io/magic-square/

---

## Features

* Generate magic squares dynamically

* Interactive and responsive interface

* Automatic calculation of the magic constant

* Visual representation of rows, columns, and diagonals

* Real-time updates

* Works directly in the browser

---

## What is a Magic Square?

A magic square of order **n** is an arrangement of numbers from:

```text
1 → n²
```

placed inside an **n × n** grid so that:

* Every row has the same sum
* Every column has the same sum
* Both diagonals have the same sum

The magic constant is calculated as:

```text
M = n(n² + 1) / 2
```

For example, a classic 3×3 magic square:

```text
8  1  6
3  5  7
4  9  2
```

Magic sum:

```text
15
```
